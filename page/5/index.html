如何在hexo中写文章

发表于 2018-05-10 | 阅读次数:

进入自己生成github页面的文件夹路径里，

命令：$ cd D:/blog/ricesgui.github.io/

命令：$ hexo new “blog1”

会在相应文件夹如我的示例D:/blog/ricesgui.github.io/source/_posts/下生成一个blog1.md文件


接着打开该md文件使用markdown语法即可书写你喜欢的博客文章




如何在hexo中建立菜单：进入自己生成的github页面的文件夹路径里，

命令：$ hexo new page “github”

会在相应文件夹如我的示例D:/blog/ricesgui.github.io/source/github/下生成一个index.md文件

编辑index.md,
随便写两句话 eg.my first page!

Eclipse报错Tomcat服务器

发表于 2018-04-01 | 阅读次数:


启动Tomcat服务器出错：Several ports (8005, 8080, 8009) required by Tomcat v9.0 Server at localhost are already in use. The server may already be running in another process, or a system process may be using the port. To start this server you will need to stop the other process or change the port number(s).


在Eclipse内已提前配置好，此处报错原因是端口（8005，8080，8009）被占用，可能原因是因为我们自己手动开了Tomcat，这时候尝试了手动关掉Tomcat，一般情况下是就已经可以了，但是再去运行发现还是不行，这时候就需要打开任务管理器，杀掉tomcat的进程。

解决方法：打开任务管理器，找到Tomcat9.exe，点击并结束进程。

一、安装node.js环境

1.在官网：https://nodejs.org/en/ 选择安装包node-v9.8.0-x86.msi(在本处可以选择适合自己电脑的版本，由于博主电脑是32位的所以选择X86，64位的安装包可以在download里找到，不再细讲)，
2.点击安装包，一路保持默认设置安装即可。

然后打开命令提示符，输入node-v ,若出现版本号，则说明安装成功。
